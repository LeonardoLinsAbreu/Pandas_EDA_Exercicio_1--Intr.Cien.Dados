# Introdução à Ciência de Dados - Atividade EDA Pandas

Este repositório contém a atividade solicitada da disciplina de "Introdução à Ciência de Dados".
O objetivo foi analisar um "DataFrame" contendo dados sobre emendas, no qual, aplicamos diversas funções básicas existentes na biblioteca Pandas.

**Conteúdo**

- Análise de um DataFrame com dados fictícios
- Utilização de funções de análise (ex.: describe(), df.head(), df.shape, df.columns, df.isnull.sum(), df.dtypes, df.sort_values().head(), df.tail() e df.duplicated().sum())
- Visualização simples dos dados
- Versionamento do projeto para o GitHub

**Dependências**

- Python
- Pandas
- VS Code
- Git/GitHub
